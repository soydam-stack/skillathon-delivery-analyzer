# 배달 주문 내역 분석기

배달앱 주문 내역 CSV를 붙여넣으면 식비·주문 습관·요일 패턴·음식 카테고리·
월별 비교·절약 목표를 한 페이지 리포트로 정리해주는 Codex Skill입니다.

## 문제 정의

"이번달 배달비 얼마나 썼지?", "나 치킨 너무 많이 시키는 거 아니야?"
— 배달앱 3개를 쓰면 지출이 분산되어 전체 파악이 어렵습니다.
이 스킬은 CSV 한 장으로 한 달 치 배달 소비를 한눈에 정리합니다.

**대상 사용자**: 배달앱을 자주 쓰는 한국인 누구나

이 저장소는 실제 주문 내역이나 개인정보를 포함하지 않고, 공개 가능한 mock data로
같은 분석 흐름을 재현합니다.

## 실행 방법

Python 3.8 이상, 외부 패키지 설치 없음.

```bash
git clone <repo-url>
cd skillathon-delivery-analyzer
python3 analyze.py
cat result.md
```

예산 목표를 바꿔 분석할 수도 있습니다.

```bash
python3 analyze.py --budget 350000 --output result.md
```

## 결과물 예시

```
# 2025년 04월 배달 주문 분석 리포트

> 이번달 **BBQ치킨**을(를) 5번 시켰고 총 **428,500원**을 배달앱에 썼습니다. 🐔

## 💰 이번달 식비
| 항목 | 금액 |
|------|------|
| 총 식비 | 428,500원 |
| 주문 횟수 | 25회 |
...

## 🏷️ 음식 카테고리 분석
| 카테고리 | 주문 횟수 | 지출 | 비율 |
|----------|-----------|------|------|
| 치킨 | 9회 | 186,000원 | 43% |

## 🎯 절약 목표
| 항목 | 값 |
|------|----|
| 월 예산 | 400,000원 |
| 현재 지출 | 428,500원 |
```

## 한계

- 실제 배달앱 API 연동 없음. CSV 직접 입력 필요.
- 월별 비교는 CSV 안에 여러 달 데이터가 있을 때 자동으로 더 의미 있게 동작.
- 금액 컬럼은 쉼표 없는 숫자만 허용.
