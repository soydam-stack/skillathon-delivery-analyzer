# Codex 실행 프롬프트 예시

## 기본 실행

```
orders.csv를 읽어서 이번달 배달 주문을 분석하고 result.md를 만들어줘.
analyze.py를 실행하면 돼.
```

## 특정 인사이트 요청

```
result.md를 읽고 "내가 배달비를 줄이려면 어떻게 해야 할지"
구체적인 조언을 3가지 알려줘.
```

## 데이터 추가 후 재분석

```
orders.csv에 아래 주문을 추가하고 analyze.py를 다시 실행해줘:
2025-04-30 20:00,배달의민족,BBQ치킨,황금올리브치킨,22000
```

## 검증 포함 실행

```
analyze.py를 실행하고 SKILL.md의 검증 명령까지 실행해서
모든 항목이 통과하는지 확인해줘.
```
