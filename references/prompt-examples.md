# Codex 실행 프롬프트 예시

## 기본 실행

```
orders.csv를 읽어서 이번달 배달 주문을 분석하고 result.md를 만들어줘.
analyze.py를 실행하면 돼.
```

## 특정 인사이트 요청

```
result.md를 읽고 "내가 배달비를 줄이려면 어떻게 해야 할지"
구체적인 조언을 3가지 알려줘.
```

## 데이터 추가 후 재분석

```
orders.csv에 아래 주문을 추가하고 analyze.py를 다시 실행해줘:
2025-04-30 20:00,배달의민족,BBQ치킨,황금올리브치킨,22000
```

## 검증 포함 실행

```
analyze.py를 실행하고 SKILL.md의 검증 명령까지 실행해서
모든 항목이 통과하는지 확인해줘.
```

## 예산 목표 포함 실행

```
orders.csv를 읽어서 월 배달비 목표 35만원 기준으로 분석해줘.
python3 analyze.py --budget 350000 --output result.md를 실행하면 돼.
```

## 절약 인사이트 해석

```
result.md의 절약 목표와 절약 인사이트 섹션을 읽고
다음 달에 바로 실천할 수 있는 행동 계획 3가지를 알려줘.
```

## 여러 달 데이터 비교

```
orders.csv에 3월과 4월 주문이 같이 들어 있어.
월별 비교 섹션에서 어떤 달에 지출이 늘었는지 설명해줘.
```
