# orders.csv 데이터 스키마

## 컬럼 정의

| 컬럼 | 타입 | 형식 | 예시 |
|------|------|------|------|
| 주문일시 | string | YYYY-MM-DD HH:MM | 2025-04-01 19:32 |
| 앱 | string | 배달의민족/쿠팡이츠/요기요 중 하나 | 배달의민족 |
| 음식점 | string | 자유 텍스트 | BBQ치킨 |
| 메뉴 | string | 자유 텍스트 | 황금올리브치킨 |
| 금액 | integer | 숫자만, 쉼표 없이 | 22000 |

## 주의사항

- 금액에 쉼표(22,000) 입력 시 파싱 오류 발생 → 22000으로 입력
- 주문일시 초(秒) 단위는 지원하지 않음 → HH:MM까지만
- UTF-8로 저장할 것 (Excel에서 내보낼 때 UTF-8 with BOM 주의)
